# Code of Conduct

## Our Commitment

EcoNeTool is a scientific project developed as part of the MARBEFES HORIZON EUROPE initiative. We are committed to maintaining a professional, welcoming, and productive environment for all contributors and users.

## Expected Behavior

All participants in the EcoNeTool community are expected to:

- Be respectful and considerate in all communications
- Focus discussion on the scientific and technical merits of the work
- Welcome newcomers and help them get started
- Give and receive constructive feedback gracefully
- Acknowledge contributions from others
- Use inclusive language

## Unacceptable Behavior

The following behaviors are not tolerated in any project space (issues, pull requests, discussions, or other communications):

- Personal attacks or insulting comments
- Publishing others' private information without permission
- Any conduct that would be inappropriate in a professional scientific setting

## Scope

This Code of Conduct applies to all project spaces including the GitHub repository, issue tracker, pull requests, and any other forums maintained by the project.

## Reporting

If you observe behavior that violates this Code of Conduct, please report it by opening a private communication with the project maintainers. All reports will be reviewed and addressed as appropriate.
